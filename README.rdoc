== Dimensional Campaign Finance

政治獻金資料分析 http://dcf.ihower.tw

Chrome 建議安裝 JSON Formatter extension: https://github.com/callumlocke/json-formatter

=== JSONP

每個 URL 都可以加上 callback 參數成為 JSONP，例如 http://dcf.ihower.tw/accounts?callback=foo
